# Library Management System
Kutubxona boshqaruv tizimi — bu kutubxonadagi kitoblarni boshqarish uchun mo‘ljallangan dastur. Dastur yordamida yangi kitob qo‘shish, barcha kitoblarni ko‘rish, kitobni ijaraga berish va qaytarib olish mumkin. Shuningdek, qaysi kitob band yoki bo‘sh ekanini ham ko‘rsatadi. 📚

## **Asosiy kommandalar**
1. **Kitob qo'shish**
2. **Kitob ijaragabersh**
3. **Barcha kitoblar**
4. **Kitob qaytarish**
5. **Chiqish**

## 1.1 **Kitob qo'shish**
Consoledan 1ni tallasa unda kitob qoshish 📖 munkun
u uchun kitobga id kiritib , Titlega nomini bering , Authorga mualifni bering shundan sonig kitob qoshiladi.

## 2.1 **Kitob ijaragabersh**
Consoledan 2ni tanlasa qaysi kitopni ijaraga olishni suraydi 
va mijozdan id,Title,Authorni kiddirishini suraydi va u kitobni ijaraga beradi.
